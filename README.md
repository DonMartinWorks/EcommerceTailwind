
# Proyecto

Este proyecto es un template de ecommerce


## Installation

Install my-proyecto with npm

```bash
  npm install
  cd proyecto
```

## Servidores

Levantar servidor NPM

```bash
  npm run dev
  cd proyecto
```

El servidor se levanta utilizando live server con el puerto: http://127.0.0.1:5500/
